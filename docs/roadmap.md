# Initial Roadmap - Genesis Engine

## Short-Term Goals

1. Define the project vision and core
   - Structure basic documentation
   - Create a centralized development log file

2. Build the RAG prototype
   - Implement document ingestion in Python
   - Create a simple vector store for semantic search
   - Separate technical and business planning knowledge bases

3. Establish the agent workflow
   - Define how Claude Code queries the RAG
   - Ensure the agent updates the development log
   - Create usage examples for tasks and decisions

## Mid-Term Goals

1. Integrate real data sources
   - Skills.sh, ADRs, technical manuals, company documents
   - Create continuous ingestion pipelines

2. Automate business planning
   - Mission, vision, values, target market, product
   - Generate RAG-based strategies and OKRs

3. From code to production
   - Generate code from planned tasks
   - Record deploys and pipelines in the log
   - Implement automated agent reviews

## Long-Term Goals

1. Multi-project support
   - Reuse the same engine across different companies or products
   - Configure separate contexts for each client or initiative

2. Specialized agents
   - Strategic planning agent
   - Software engineering agent
   - Operations / deployment agent

3. Continuous evolution
   - Retrievable project history memory
   - Learning and refinement of internal strategies
